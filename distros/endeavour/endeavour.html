Guia de Instalação do EndeavourOS

EndeavourOS é uma distribuição baseada em Arch Linux, voltada para usuários que querem um sistema Arch pronto para uso, mas ainda flexível. Este guia ajuda a instalar passo a passo.

Requisitos Mínimos

  • 2 GB de RAM (4 GB recomendado)
  • 20 GB de armazenamento
  • Processador de 64 bits
  • Pen drive de 4 GB ou mais

Passos da Instalação

  1. Baixe a ISO oficial do EndeavourOS.
  2. Crie um pendrive bootável com Rufus ou Balena Etcher.
  3. Inicialize o computador pelo pendrive.
  4. Escolha “Install EndeavourOS”.
  5. Configure idioma, teclado e rede.
  6. Escolha particionamento e finalize a instalação.
  7. Reinicie o computador.
