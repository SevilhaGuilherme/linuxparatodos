Guia de Instalação do Manjaro

Manjaro é uma distribuição baseada no Arch Linux, mas mais amigável para iniciantes, oferecendo um sistema pronto para uso. Siga este guia para instalar corretamente.

Requisitos Mínimos

  • 1 GB de RAM (2 GB recomendado)
  • 30 GB de armazenamento
  • Processador de 64 bits
  • Pen drive de 4 GB ou mais

Passos da Instalação

  1. Baixe a ISO oficial do Manjaro.
  2. Crie um pendrive bootável com Rufus ou Balena Etcher.
  3. Inicialize o computador pelo pendrive.
  4. Escolha “Install Manjaro”.
  5. Configure idioma, teclado e rede.
  6. Escolha o tipo de instalação (limpa ou dual boot).
  7. Finalize a instalação e reinicie o computador.
